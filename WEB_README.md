# 新闻 Web 应用使用说明

## 快速启动

### 方法 1：使用启动脚本（推荐）

```bash
./start_web.sh
```

### 方法 2：直接运行

```bash
# 安装 Flask（如果还没安装）
pip install flask

# 启动服务器
python web_server.py
```

## 访问地址

启动后在浏览器访问：**http://localhost:5000**

## 功能特性

### 📰 新闻列表
- 精美的卡片式布局
- 渐变色背景设计
- 响应式设计，支持移动端
- 显示新闻标题（中英双语）
- 显示来源、分类、可信度评分
- 星级可信度展示

### 🔍 筛选功能
- **按来源筛选**：选择特定新闻源
- **按分类筛选**：科技、政治、财经等
- **按可信度筛选**：高可信度、中等可信度
- **按时间筛选**：今天、最近3天、7天、30天

### 📄 翻页功能
- 每页显示 20 条新闻
- 智能分页导航
- 显示总新闻数量
- 保持筛选条件

### 📖 新闻详情
- 点击新闻卡片查看详情
- 显示完整的原文和中文翻译
- 显示验证标签和警告信息
- 可信度详细评分
- 原文链接跳转

## 界面预览

### 列表页面
- 渐变紫色背景
- 白色卡片设计
- 悬停动画效果
- 清晰的信息层级

### 详情页面
- 大标题展示
- 中英双语对照
- 验证信息突出显示
- 警告信息醒目提示

## 技术栈

- **后端**：Flask (Python)
- **前端**：HTML5 + CSS3
- **数据库**：SQLite
- **样式**：现代渐变设计 + 响应式布局

## 配置

默认配置：
- 端口：5000
- 每页显示：20 条新闻
- 主机：0.0.0.0（允许外部访问）

修改配置请编辑 `web_server.py` 文件中的常量。

## 注意事项

1. 确保已经抓取了新闻数据（运行 `python main.py fetch`）
2. 数据库文件位置：`data/news.db`
3. 如需修改端口，编辑 `web_server.py` 最后一行

## 停止服务器

在终端按 `Ctrl + C` 停止服务器。
